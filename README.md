# Real-Time-Oil-Spill-Detection-Using-AIS-Satellite-Imagery
AI-powered oil spill detection using Sentinel-1 SAR imagery &amp; AIS vessel data. Utilizes UNet, DeepLabV3+, and a hybrid model (>92% IoU) for semantic segmentation. Real-time capable, scalable, and applicable to environmental monitoring, law enforcement, research, and pollution control.
